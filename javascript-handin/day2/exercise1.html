Colorful Divs

 
 
 

Description of Solution

To solve this task, we created an HTML page with three <div> elements, each assigned a unique ID: d1, d2, and d3. These <div> elements were initially given different background colors as specified in the task.

We also added a button with the ID "colorButton" to change the background color of the three <div> elements when clicked.

For styling purposes, we defined CSS rules in the <style> section to style the header, <div> elements, and the button. JavaScript was used to add functionality to the button, changing the background colors of the <div> elements to random colors upon clicking.
